# Table of keybinds

This is the table of all possible keybinds in our project

| Key          | Action         |
| ------------ | -------------- |
| <kbd>h</kbd> | Open help menu |
| <kbd>w</kbd> | Go up          |
| <kbd>s</kbd> | Go down        |
| <kbd>z</kbd> | Delete event   |
| <kbd>a</kbd> | Add event      |
| <kbd>q</kbd> | Exit           |
